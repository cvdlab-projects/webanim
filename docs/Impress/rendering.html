Rendering Sub-Project
Developed by

Davide Ceneda and Alessio Baroni
-->
The purpose of this subproject was to animate 3D objects chosen by the user in the previous initialization phase. To achieve this goal we used two javascript libraries: Three.js and Tween.js.

Three js render meshes while Tween js take care of positioning, rotating and scaling them on the canvas

Moreover we added a lot of other features not included in the original library such as:
STOP, REWIND, FAST FORWARD, CAMERAS ANIMATION, START FROM SECOND, START FROM FRAME, SWITCH BETWEEN CAMERAS.
